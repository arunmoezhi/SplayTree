Insert k v 	 - Inserts a key value pair and then splays
Find k     	 - Splays for key 'k' and then searches for it
Remove k 	 - Splays for key 'k' and then removes it
RemValue v	 - Removes all keys with value v
Min			 - Find the minimum key and return its value
Max 		 - Find the maximum key and return its value
Size		 - Find the number of keys in the tree
Output		 - Sum of values returned by Min, Max, Size and RemValue
